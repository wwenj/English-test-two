## 功能实现
* **首页:**
* **判断题:** 必选只能选择一项，点击下一题把本题答案post提交到后台
* **单选填空:** 必选只能选择一项，点击下一题把本题答案post提交到后台
* **多填空:** 把答案按顺序填到空处，必须填满，点击下一题把本题答案post提交到后台
* **评价星级:** 全部题答完后根据后台返回的答对题的个数展示相应的星级
